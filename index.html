Aplikasi enkripsi file lokal dengan antarmuka grafis modern. Melindungi file Anda menggunakan standar enkripsi militer AES-256-GCM.
🔐
AES-256-GCM
Enkripsi standar militer dengan autentikasi data bawaan.
🔑

Derivasi kunci yang kuat, tahan serangan brute force.
📁
Rekursif
Memproses semua file dalam folder dan sub-folder secara otomatis.
🛡️
Salt & IV Unik
Setiap file dienkripsi dengan salt dan IV acak yang berbeda.
